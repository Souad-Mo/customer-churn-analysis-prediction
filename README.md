# Customer Churn Analysis & Prediction

## Overview
Customer churn analysis and prediction using Logistic Regression, Random Forest, and XGBoost on the IBM Telco dataset (~7,000 records).

## Key Findings
- Churn rate is ~27%.
- Contract type, tenure, and monthly charges are the strongest churn predictors.
- Logistic Regression recommended as best model (81% accuracy, highest churn recall).

## Business Recommendations
- Target month-to-month customers with contract incentives.
- Improve onboarding for new customers.
- Bundle security and tech support services.
- Investigate fiber optic service quality.
